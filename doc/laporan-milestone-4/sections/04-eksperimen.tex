\section{Strategi Pengujian}
Pengujian dilakukan dengan membangun program menggunakan \texttt{make -B}, menjalankan \texttt{arion\_parser} pada input Milestone 4, lalu membandingkan output aktual dengan hasil eksekusi yang diharapkan. Perlu dicatat bahwa \texttt{arion\_parser} menjalankan seluruh pipeline compiler secara berurutan, mulai dari lexical analysis, syntax analysis, semantic analysis, intermediate code generation, hingga eksekusi oleh interpreter. Akibatnya, setiap test case yang dijalankan secara otomatis memvalidasi kebenaran Milestone 1, Milestone 2, Milestone 3, dan Milestone 4 sekaligus. Command yang digunakan adalah sebagai berikut.

\begin{tcolorbox}[title=Command Pengujian,colback=gray!5,colframe=black!60,boxrule=0.5pt,arc=2mm]
  \begin{verbatim}
make -B
for i in 1 2 3 4 5 6 7; do
  ./arion_parser test/milestone-4/input-m4-$i.txt output-m4-$i.txt
  echo "Test $i exit code: $?"
done
\end{verbatim}
\end{tcolorbox}

Seluruh test case yang digunakan berhasil menghasilkan output intermediate code dan interpreter output yang sesuai dengan ekspetasi. Laporan ini menampilkan tujuh kasus yang mewakili operasi dasar, control flow, perulangan, case statement, array, dan runtime error.

\section{Pengujian Lintas Milestone}
Karena \texttt{arion\_parser} menjalankan pipeline secara utuh, setiap test case pada Milestone 4 sekaligus menjadi bukti kebenaran milestone-milestone sebelumnya. Bagian Token Stream dalam output membuktikan Milestone 1 telah berhasil mengenali seluruh token secara tepat, mulai dari keyword, identifier, operator, hingga delimiter. Bagian Parse Tree membuktikan Milestone 2 telah berhasil membangun struktur hierarki program sesuai grammar Arion. Bagian \texttt{tab}, \texttt{btab}, \texttt{atab}, dan Decorated AST membuktikan Milestone 3 telah berhasil melakukan analisis semantik, termasuk pengecekan tipe, scope, dan deklarasi. Bagian Intermediate Code dan Interpreter Output membuktikan Milestone 4 telah berhasil meratakan decorated AST menjadi instruksi sekuensial dan mengeksekusinya dengan benar.
